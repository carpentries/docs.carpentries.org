# Etiquetas de problemas de GitHub

The Carpentries utiliza un conjunto ampliado de etiquetas de emisión. Estas etiquetas se pueden aplicar a los problemas de un repositorio para facilitar la categorización y la priorización y facilitar a los visitantes del repositorio la comprensión más rápida del estado de un problema determinado.
Se anima a los mantenedores a utilizar estas etiquetas y a añadir etiquetas adicionales en sus repositorios si las encuentran útiles.
