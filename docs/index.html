This is a prototype of the new Carpentries Handbook and does not contain current information. For more information please visit The Carpentries main website or The Carpentries current handbook. The Carpentries teaches foundational coding and data science skills to researchers worldwide. Software Carpentry, Data Carpentry, and Library Carpentry workshops are based on our lessons. Workshop hosts, Instructors, and learners must be prepared to follow our Code of Conduct.
